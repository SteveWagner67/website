Project 1: Tracer


Goal

To know the placement of a parcel in real time through an embedded system

Method

The user enters the arrived coordinate in a text file with his phone number, with which it will be become a SMS when the parcel is arrived at destination. The textfile will be saved into a server. The embedded system receives his GPS's coordinate every 5 minutes through a GPS module and send this coordinate to the server. The user can also check the parcel's position when he wants to. The server compares the received coordinate with the saved coordinate. If the coordinates correspond, the server send a message with the user's phone number to the embedded system. The embedded system send at this time a SMS to the user to inform him that his parcel is arrived.

Team

2 Students in Bachelor of Informatic specialised in embedded system

Programmation Software

MBED plateform, Waspmote plateform

Programmation Language

C, C++

Material used

Ublox C027 and Wapsmote embedded board

Time of work

50 heures (Février 2015 – Avril 2015)

Final Result

The embedded systems receive their GPS's coordinate and send them to the server. A SMS is sent to the user when the coordinate correspond

Project 2: Autonom Robot Car


Goal

Pouvoir connaitre le placement d’un colis grâce à un système embarqué.

Method

Utilisation de capteurs ultrasoniques pour déterminer les distances de part et d’autres du robot. Celui-ci se déplace de manière automatique et évite les obstacles. Depuis l’interface utilisateur, on peut choisir de laisser le robot se déplacer seul ou choisir de le guider nous-même. On peut voir la distance de chaque capteur et où se trouve un obstacle.

Team

Only

Programmation Software

Codewarrior, LabVIEW

Programmation Language

C

Time of work

150 hours (Frebruary 2015 - May 2015)

Final Result

Le robot se déplace de manière automatique. On peut le commander depuis l’interface utilisateur (choix des vitesses, commande du départ et de l’arrêt, choix du mode automatique ou manuel, possibilité de le guider nous-même)
